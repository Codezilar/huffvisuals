## 🚀 Getting Started (3 Steps)

### Step 1: View Your Site (30 seconds)
```bash
# Just open the file
open index.html

# Or use a local server
python -m http.server 8000
# Visit: http://localhost:8000
```

### Step 2: Customize Content (5 minutes)
Open `index.html` and edit:
- Brand name (search for "HUFF")
- Title and description
- Social media links
- Portfolio items
- Exhibition details

### Step 3: Deploy (5 minutes)
Choose one platform:
- **Netlify** - Easiest, recommended
- **Vercel** - Fast, similar to Netlify
- **GitHub Pages** - Free, simple
- **Your hosting** - FTP upload

See `DEPLOYMENT.md` for detailed instructions.

## 🐛 Troubleshooting

### Site looks broken
→ Make sure all files are in same folder

### Links don't work
→ Check that href starts with `#` for internal links

### Images not showing
→ Replace SVG placeholders with real image files

### Mobile menu stuck
→ Clear browser cache (Ctrl+Shift+Delete)

### Form not sending
→ Setup with Formspree or EmailJS (see DEPLOYMENT.md)

### Performance issues
→ Optimize images, check DevTools Lighthouse
